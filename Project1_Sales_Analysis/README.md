# Project 1: Sales Analysis Dashboard

## Objective
To analyze sales data from an online store to identify top-performing products, monthly sales trends, and customer purchase patterns.

## Tools Used
- **Python:** Pandas (data cleaning), Matplotlib/Seaborn (exploratory analysis).
- **Power BI:** Interactive dashboard creation.

## Process
1. **Data Cleaning:** Used Python Pandas to handle missing values, remove duplicates, and format date columns.
2. **Exploratory Data Analysis (EDA):** Created visualizations to understand sales distribution by category and region.
3. **Dashboard Development:** Built an interactive Power BI dashboard with key metrics (total revenue, average ticket) and filters for dynamic exploration.

## Key Insights (Ejemplo de lo que podrías encontrar)
- Electronics category generates 40% of total revenue.
- Sales peak in November and December (holiday season).
- Top 10% of customers account for 30% of sales.

## Files
- `notebooks/sales_analysis.ipynb`: Jupyter notebook with Python code.
- `powerbi/sales_dashboard.pbix`: Power BI dashboard file.
- `data/`: Folder containing the dataset (or link to source).
