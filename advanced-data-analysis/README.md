🎬 Movie Data Analysis

📌 Project Overview

This project explores a movie dataset to analyze relationships between budget, revenue, ratings, and popularity. The goal is to uncover trends within the film industry using data analysis techniques.

---

🎯 Objectives

- Analyze the relationship between budget and revenue
- Identify factors influencing movie success
- Explore rating trends and audience preferences

---

🛠️ Tools & Technologies

- Python (pandas, matplotlib)
- SQL
- Jupyter Notebook

---

📂 Dataset

Source: (Kaggle / public dataset)

Description:

- Includes movie metadata such as budget, revenue, genre, rating

---

🔍 Data Analysis Process

1. Data cleaning and preprocessing
2. Exploratory Data Analysis (EDA)
3. SQL queries for segmentation
4. Visualization of trends and relationships

---

📈 Key Insights

- Higher budgets tend to correlate with higher revenues, but not always
- Certain genres consistently outperform others
- Audience ratings do not always align with financial success
